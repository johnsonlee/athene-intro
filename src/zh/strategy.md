## 从常识到策略

*三个逻辑如何变成选股方法*

----

## 三个逻辑 → 三个问题

每次选股前，只需要回答三个问题：

1. **这个价格值不值？**
2. **公司在变好还是变坏？**
3. **现在是好时机吗？**

> 三个问题，缺一不可

----

<!-- .slide: class="center" -->

## 问题一

### 这个价格值不值？

*推导自：逻辑二（钱不会凭空消失）+ 逻辑三（变化是渐进的）*

----

## 估值空间 (VM)

**逻辑二**告诉我们：资金轮动会制造错误定价 → 有些股票被低估了

**逻辑三**告诉我们："便宜"不等于"值得买"

- 一家盈利在**增长**的公司，PE 高一点是合理的
- 一家盈利在**下降**的公司，PE 再低也可能是陷阱

> Athene 的做法：用**增长率调整估值**——成长型给溢价，衰退型打折扣

----

<!-- .slide: class="center" -->

## 问题二

### 公司在变好还是变坏？

*推导自：逻辑二（钱不会凭空消失）*

----

## 盈利可见度 (EV)

**逻辑二**告诉我们：聪明钱的流动方向是最真实的判断

- 分析师的评级调整 = 机构资金即将流动的信号
- 盈利增长的趋势 = 公司基本面的方向
- 两者结合 = **聪明钱认为这家公司的未来如何**

----

## 方向系数：估值的放大镜

方向系数是盈利可见度的**乘数**——

- 分析师上调 + 盈利增长 → 系数 1.2×~1.5× → **放大估值吸引力**
- 分析师不动 + 盈利平稳 → 系数 ≈ 1× → **保持原样**
- 分析师下调 + 盈利下滑 → 系数 0.2×~0.5× → **大幅缩水**

> 最极端情况：方向系数压到 0.2×，EV 几乎归零——乘法模型里，这意味着总分也接近零
>
> 和择时一样：**看不到方向的公司，不值得下注**

----

<!-- .slide: class="center" -->

## 问题三

### 现在是好时机吗？

*推导自：逻辑一（趋势不是随机的）+ 逻辑三（变化是渐进的）*

----

## 择时 (Timing)

**逻辑一**告诉我们：趋势反映周期位置

- 强势上涨 = 周期后期 = 剩余空间少
- 弱势下跌 = 周期早期 = 潜在空间大

**逻辑三**告诉我们：变化是渐进的

- 下跌中的"空间大"可能是陷阱，也可能是机会
- 必须等到**反转信号确认**才行动

----

## 反转信号是什么？

不需要记住复杂的公式，只需要理解一个概念：

**反转信号 = 跌势开始减弱的证据**

- 就像钟摆，到达最高点前速度会减慢——趋势接近极端时，反转的力量在积聚
- Athene 追踪四种反转迹象：
  - 下跌动能减弱
  - 超卖后的回升
  - 价格靠近支撑位
  - 短期指标出现拐点

> 没有反转信号 → 择时得分 = 0 → **不行动**

----

<!-- .slide: class="center" -->

## 为什么三者缺一不可？

*推导自：逻辑三（变化是渐进的）*

----

## 三脚架的比喻

想象一个**三脚架**——

- 抽掉"估值"这条腿 → 在天价买入好公司
- 抽掉"可见度"这条腿 → 买了便宜货但不知道会不会继续烂
- 抽掉"择时"这条腿 → 好公司好价格，但买早了三个月照样亏

> Athene 用**乘法**而不是加法：任何一个因子为零，总分就是零

----

## 加法 vs 乘法

| | 传统加法模型 | Athene 乘法模型 |
|---|---|---|
| 逻辑 | 优点可以弥补缺点 | 缺一不可 |
| 风险 | 可能买到"瘸腿"股票 | 三条腿都要站稳 |
| 类比 | 考试算总分：偏科也能过 | 面试三轮：每轮都要过 |

> **逻辑三**告诉我们：周期是连续的，所有条件必须同时满足才安全

----

## 完整推导链

| 逻辑 | 推导出 | 对应因子 |
|------|--------|----------|
| 趋势不是随机的 | 趋势反映周期位置 → 等待反转确认 | 择时 (Timing) |
| 钱不会凭空消失 | 追踪聪明钱 → 盈利方向 + 错误定价 | 盈利可见度 (EV) + 估值空间 (VM) |
| 变化是渐进的 | 强势=晚期, 弱势=早期 → 逆向 + 乘法 | 周期反转 + 缺一不可 |

> **三个朴素的逻辑 → 三个核心问题 → 三个评分因子 → 一套乘法公式**

----

| 问题 | 因子 | 源自哪个逻辑 |
|------|------|-------------|
| 价格值不值？ | 估值空间 (VM) | 资本零和 + 周期连续 |

| 时机对不对？ | 择时 (Timing) | 趋势预测 + 周期连续 |
